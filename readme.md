Conceitos e sintaxe do Typescript
- String
- Number
- Boolean
- Array
- Tupla
- Enum
- Any
- Unknown
- Null e Undefined
- Object



Comandos inicias:

criar pacote npm = npm init -y ira cirar um package.json

iniciar typescript = npm i --save-dev typescript

instalar dependencias = npx tsc --init

script de build = npm run build
